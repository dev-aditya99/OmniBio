# 🌐 OmniBio

> A Link-in-Bio SaaS platform for social media influencers and creators.

Create your personalized bio link page in seconds. Share all your content, social profiles, and links in one place. OmniBio acts as a centralized hub for your digital identity, routing your followers exactly where you want them to go.

## ✨ Features

- **Custom Public Profile:** Clean, mobile-responsive page (`/username`) for your followers.
- **Creator Dashboard:** Add, edit, reorder, and delete your links in real-time.
- **Secure Authentication:** Powered by Firebase Auth (Email/Password & Google Sign-In).
- **Fast & SEO Friendly:** Built with Next.js App Router for optimal performance.
- **Modern UI:** Designed with TailwindCSS for a sleek, professional look.

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, TailwindCSS
- **Language:** TypeScript
- **Backend/Database:** Firebase Firestore
- **Authentication:** Firebase Auth

## 🚀 Getting Started

Follow these instructions to set up the project on your local machine.

### Prerequisites

Make sure you have Node.js installed on your system.

- [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Installation

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/dev-aditya99/omnibio.git](https://github.com/dev-aditya99/omnibio.git)
   cd omnibio
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   yarn install
   ```
